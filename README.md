# Tritan EV Dock Website


EV Dock is a modern and responsive web application designed for electric vehicle (EV) charging station management and information services.

## Technologies Used
- HTML5
- React.js
- TypeScript
- Tailwind CSS

## Features

- Responsive and user-friendly interface
- Modern UI built with Tailwind CSS
- Fast and scalable React architecture
- Type-safe development using TypeScript
- EV charging station information and services
- Cross-device compatibility
